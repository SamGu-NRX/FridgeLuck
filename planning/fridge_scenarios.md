# Fridge Scenarios — Demo Mode

Each scenario pre-loads a curated set of ingredients (via bundled fixture JSON) so users can experience the full FridgeLuck flow without a real fridge photo.

---

## 1. Quick Breakfast

**Fixture:** `demo_breakfast.json`
**Icon:** `sun.horizon.fill`
**Palette:** oat

**Ingredients:** Eggs, bread, banana, oats, milk, butter

**Expected recipes:**
- French Toast
- Banana Oat Pancakes

**Description prompt:**
> A sunny morning spread — everything you need for a fast, filling breakfast. Crack some eggs, toast some bread, and see what FridgeLuck can whip up.
